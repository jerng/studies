# superails
A study of popular computing technologies, that uses Ruby on Rails for glue.

# involves
* Docker
* jRuby
* Ruby on Rails
* MySQL

# try it
* Build the Dockerfile, and run the resulting image with 
  `docker run --rm -it -p 3000:3000 TAG-OR-IMAGE-ID`
